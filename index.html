ScrollViewer v1.0 使用說明

【功能說明】
　方便瀏覽電腦中的長條圖片，尤其是長條漫畫的分割壓縮檔。
【使用方法】
　按 [本地端檔案] 選定資料夾，清單中會列出資料夾下的圖檔，
　點選清單選擇檔案瀏覽。
【檔案格式】
　支援 .bmp檔, .gif檔, .jpg檔, .png檔, .zip檔
　顯示.zip檔時，會將壓縮檔中的圖片串成長條圖，方便滾動瀏覽。
【圖示與快速鍵】
： 選單列，點擊可釘選
： 開啟資料夾 ( 快速鍵：Ctrl + O )
： 使用說明 ( 快速鍵：F1 )
： 上一話 ( 快速鍵：← )
： 下一話 ( 快速鍵：→ )
： 回到頂端 ( 快速鍵：Home )
： 放大 ( 快速鍵：+ )
： 縮小 ( 快速鍵：- )
： 原始大小 ( 快速鍵：= 或 Enter )
