# ScrollViewer2

長條圖片／漫畫瀏覽器，使用 Electron 開發。

## 功能

- 瀏覽本地資料夾中的圖片（bmp/jpg/png/gif/webp/tif/tiff）
- 開啟 zip/cbz、rar/cbr 壓縮檔，將內容串成長條圖捲動閱讀（內建壓縮炸彈防護：偵測異常壓縮比與總解壓大小）
- 含圖片的子資料夾會自動當成一話串接瀏覽，同層散落的圖片也會自動合併成一話
- 依名稱或修改時間排序，清單可搜尋／篩選章節
- 縮放：放大/縮小/原始大小、直接輸入百分比、或符合視窗寬度，間隔可自訂
- 格狀縮圖瀏覽模式（預設關閉，偏好設定中可啟用）
- 無縫章節捲動：捲動到一話的頂端或底端時，自動載入並無縫接續上一話／下一話（預設開啟，偏好設定可關閉），自動捲動閱讀時也會一併接續
- 自動捲動閱讀（速度可調）
- ↑/↓ 鍵與 PageUp/PageDown 每次捲動的距離皆可自訂，兩者分開設定：固定像素、或依本話長度的百分比
- AI 影像增強（選用）：放大檢視解析度不足的頁面時，於背景以本地 AI（waifu2x）自動加強畫質並快取結果，偏好設定中可開啟／關閉，並提示可能的效能影響
- 縮圖捲軸預覽（點擊或拖曳跳轉）
- 已讀章節自動標記
- 閱讀進度百分比／進度條（可關閉）
- 深色模式（三種背景色調可選）與可自訂主題色
- 側邊欄可完全隱藏、可切換左右側
- 閱讀區域最大寬度設定
- 最近開啟資料夾清單
- 拖曳資料夾或壓縮檔開啟
- 全螢幕沉浸式閱讀模式
- 啟動行為可選：回到最後瀏覽進度、或固定啟動資料夾
- 偏好設定頁：外觀、版面、瀏覽行為、快速鍵皆可自訂
- 使用說明頁顯示目前版本號
- 可同時執行多個視窗，各自瀏覽不同資料夾互不影響

## 開發

```bash
npm install
npm start
```

## 測試

```bash
npm test
```

跑 `src/main/lib/` 底下與 Electron 無關的純邏輯單元測試（壓縮檔解析與安全檢查、設定檔合併、資料夾清單分類、AI 增強快取）。

## 打包

```bash
node scripts/fetch-waifu2x.js   # 下載對應平台的 AI 增強引擎（選用，未執行則該功能於此次打包中不可用）
npm run dist
```

使用 [electron-builder](https://www.electron.build/) 打包成 Windows/macOS/Linux 安裝檔，輸出於 `dist/`。

## 快速鍵（預設值，可在偏好設定頁自訂）

| 按鍵 | 功能 |
| --- | --- |
| Ctrl/Cmd + O | 開啟資料夾 |
| ← / → | 上一話 / 下一話 |
| Home | 回到頂端 |
| ↑ / ↓ | 向上 / 向下捲動（距離可自訂） |
| PgUp / PgDn | 向上 / 向下翻頁（距離可自訂） |
| + / - | 放大 / 縮小 |
| = 或 Enter | 原始大小 |
| Space | 自動捲動切換（再按一次停止） |
| Ctrl/Cmd + D | 深色模式切換 |
| Ctrl/Cmd + G | 格狀瀏覽模式切換（需先在偏好設定啟用） |
| Ctrl/Cmd + B | 側邊欄完全隱藏切換 |
| F1 | 使用說明 |
| F11 | 全螢幕沉浸式閱讀 |
| Ctrl/Cmd + , | 偏好設定 |
| Esc | 關閉使用說明 / 偏好設定 |
